Express 介绍

Express 是基于Node.js平台，快速开放极简的 Web 开发框架

Express 安装

前提 node 要下载好噢~

npm install express --save

再安装 node 自动重启工具 nodemon

npm install -g nodemon

基础安装完以后

接下来简单介绍下用法
