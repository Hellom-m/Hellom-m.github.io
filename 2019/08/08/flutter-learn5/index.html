Image 支持如下几种类型的构造函数：

new Image

  • 用于从 ImageProvider 获取图像

new Image.asset

  • 使用 key 从 AssetBundle 获取图像

new Image.network

  • 从网络 URL 中获取

new Image.file

  • 从本地文件中获取

new Image.memory

  • 从 Uint8List 获取图像

如何加载静态图片及处理不同分辨率的图片？

加载项目中的静态图片步骤：

  • 在 pubspec.yaml 文件中声明图片资源的路径

    assets:
   - images/my_icon.png

  • 使用 AssetImage 访问图片

为了让 Image 能够根据像素密度自适应不同分辨率 使用 AssetImage 指定图像 并且确保在 Widget 树中的 Image Widget 上方存在 MaterialApp 、 WidgetsApp 或 MediaQuery 窗口Widget
